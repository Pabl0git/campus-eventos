# Campus Eventos

Proyecto académico desarrollado en React para gestionar eventos universitarios, participantes e inscripciones.

## Funciones principales

- Registrar, visualizar, buscar, actualizar y eliminar eventos.
- Registrar estudiantes y docentes.
- Inscribir participantes y controlar cupos.
- Interfaz responsive para computadora, tableta y celular.
- Consumo de una API REST mediante Axios.
- Pruebas unitarias con Jest y React Testing Library.

## Tecnologías

- React y Vite
- React Router
- Axios
- Bootstrap y CSS
- JSON Server
- Jest y React Testing Library

## Instalación

1. Instalar Node.js 18 o superior.
2. Abrir una terminal dentro del proyecto.
3. Ejecutar:

```bash
npm install
npm run dev
```

La aplicación se abrirá en `http://localhost:5173` y la API se ejecutará en `http://localhost:3001`.

## Pruebas

```bash
npm test
npm run build
```

## Estructura

```text
src/components     Componentes reutilizables
src/services       Integración con la API
src/test           Configuración de pruebas
```
